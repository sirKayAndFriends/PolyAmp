# An Amp With All the Trimmings

The PolyAmpSupreme contains all of the modules in this project and is intended for embedded systems like Daisy, Rasperry Pi or whatnot.  This is also almost identical to the [VST]!

## Modules
Boost 
Preamp
PowerAmp
Tonestack
Cabsim

## Controls
Boost, Bass Cut, Frequency _OR_ Bass, Mid, Treble (see documentation section on Boost)

Overdrive

Treble, Mid, Bass, Voice

Volume

Channel - 3 way switch

Cabsim - 3 way switch
