# A Stripped Down Simpler Project

The PolyAmpCombo has all of the tone and distortion controls of the Supreme without the boost or cabsim section.  This is intended more as an effects pedal or even a physical amp!  Plug it into a power amp and see what happens!

## Modules
Preamp
PowerAmp
Tonestack

## Controls
Overdrive

Treble, Mid, Bass, Voice

Volume

Channel - 3 way switch
